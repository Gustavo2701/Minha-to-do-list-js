# Minha-to-do-list-js
To-do list com JavaScript puro, com funcionalidades de adicionar, editar, excluir e filtrar tarefas usando localStorage.

# To-do List

Projeto de lista de tarefas feito com HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar tarefas
- Editar tarefas
- Marcar como concluída
- Excluir tarefas
- Filtro (todas, pendentes, concluídas)
- Salvamento no localStorage
